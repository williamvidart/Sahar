# Dome Matarazzo

Proposta arquitetônica para um templo sonoro dentro da Cidade Matarazzo, São Paulo.

**Sahar Curation × Cidade Matarazzo**

Apresentação interativa em HTML — projeto, referências de arquitetura (Azulik), elementos construtivos, materiais, programação e estrutura de parceria.

## Stack

- HTML / CSS / JavaScript vanilla
- Tipografia: Cormorant Garamond + Inter (Google Fonts)
- Sem dependências externas além das fontes

## Como rodar localmente

```bash
python3 -m http.server 4848
# abra http://127.0.0.1:4848
```

## Navegação

- `← →` ou `Espaço` ou `scroll` — avança/retrocede
- `A B C D` — pula para o pathway correspondente
- `Esc` — volta para o fluxo principal
- `Home / End` — primeira / última cena
